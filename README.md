## О проекте

Это сайт компании **B2B Движение** — платформы для перевода оптовых продаж в онлайн. Компания помогает дистрибьюторам, оптовикам и производителям запускать B2B-порталы самообслуживания с бесшовной интеграцией в 1С.

Сайт рассказывает о возможностях платформы, демонстрирует реальные кейсы клиентов и служит точкой входа для новых заказчиков.

## Технологии

| Категория | Стек |
|---|---|
| **Фреймворк** | Next.js 16 (App Router) |
| **UI** | React 19 + Tailwind CSS 4 |
| **Язык** | TypeScript |
| **Компоненты** | shadcn/ui + Radix UI |
| **Контент** | Markdown (gray-matter) |
| **Деплой** | Vercel |

## Что здесь интересного

- **Динамическая навигация** — табы, фильтры и чипы с сохранением состояния в URL
- **Тёмная и светлая темы** — полная поддержка через `next-themes`
- **Адаптивный дизайн** — от мобильного до десктопа
- **Оптимизированные изображения** — `next/image` с приоритетной загрузкой
- **SEO-оптимизация** — мета-теги, семантическая разметка, статическая генерация


## 📂 Структура

```
app/              — страницы и маршруты (App Router)
components/       — переиспользуемые компоненты
content/          — MDX-контент (блог, новости)
lib/              — утилиты и хелперы
public/           — статические файлы
  ├── animations/   — для анимации интерфейса
  ├── backgrounds/  — фоновые изображения
  ├── icons/        — иконки
  ├── images/       — обложки блога и новостей
  ├── logos/        — логотипы компаний-клиентов
  ├── mockups/      — мокапы устройств
  ├── team/         — фото команды
  └── ui-screenshots/ — скриншоты интерфейса
```
