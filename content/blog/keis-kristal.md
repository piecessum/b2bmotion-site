## Специфика отрасли «Электротехника»

Электротехника — отрасль с **огромным номенклатурным разнообразием**. Десятки тысяч SKU с точными характеристиками: сечение, номинальный ток, класс защиты, число полюсов. Каждый товар имеет множество технических параметров, и ошибка в подборе означает нерабочую систему.

**Производитель критичен** — ABB, Schneider Electric, IEK, EKF работают в разных ценовых сегментах, и заказчики привязаны к конкретным брендам в рамках проекта.

## Ключевые вызовы поставок электрооборудования

- **Заказ по проекту** — подбор оборудования по проектной документации с точными характеристиками
- **Комплектация объекта** — полный набор электротехники для строительства или ремонта
- **Повторные закупки** — регулярные поставки на объекты
- **Подбор аналогов** — замена недоступного оборудования на совместимые аналоги
- **Аварийные закупки** — срочный поиск для устранения аварийных ситуаций

## Как платформа решает задачи поставщика электрооборудования

### Поиск по артикулу и коду

Специалисты работают с конкретными артикулами — это основной сценарий поиска. «Автомат ABB S201», «кабель 3x2.5 ВВГ» — типичные запросы. Платформа обеспечивает точный поиск по артикулу, названию серии и характеристикам.

### Технические характеристики

Полная таблица характеристик — основа для принятия решения о закупке. Номинальный ток, сечение, число полюсов, степень защиты IP — всё на виду в карточке товара.

### Группировка по сериям

Товары сгруппированы по сериям и линейкам производителя. Это позволяет быстро найти нужную модификацию в рамках выбранной серии, не перебирая тысячи позиций.

### Сертификаты и документация

Сертификаты соответствия, пожарные сертификаты, паспорта изделий и схемы подключения — всё доступно в карточке товара. Монтажная организация скачивает документы без обращения к менеджеру.

### Фильтрация и навигация

Древовидная структура категорий (группа → подгруппа → тип → серия) и фильтры по бренду, техническим параметрам, наличию позволяют быстро находить нужные позиции в каталоге из десятков тысяч SKU.
