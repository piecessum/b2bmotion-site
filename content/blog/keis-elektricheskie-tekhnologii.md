## Специфика светотехники и электромонтажа

Светотехника — это отрасль, где визуальные характеристики товара не менее важны, чем технические. Цветовая температура, индекс цветопередачи, форма и тип рассеивателя — всё это влияет на конечный результат освещения объекта. Для профессионалов электромонтажа критичны точные параметры: мощность, световой поток, IP защиты и тип монтажа.

Сеть магазинов добавляет ещё один слой сложности — клиент хочет знать, в какой именно точке есть нужный товар, чтобы забрать его максимально быстро.

## Сценарии закупок для электромонтажников

- **Комплектация объекта** — подбор всего освещения для квартиры, офиса или коммерческого помещения
- **Заказ по проекту** — точные позиции светильников и электрооборудования из проектной документации
- **Быстрая покупка расходников** — кабель, автоматы, розетки — повседневные закупки монтажных бригад
- **Подбор замены** — когда проектная позиция снята с производства или отсутствует

## Что важно для профессионалов электромонтажа

### Остатки по магазинам сети

Монтажник работает на объекте и ему нужен конкретный товар прямо сейчас. Видеть остатки по каждому магазину сети — значит точно знать, куда ехать за материалом, без звонков и уточнений.

### Специфические фильтры для светотехники

Стандартных фильтров «бренд + цена» для светотехники недостаточно. Нужны фильтры по мощности, цветовой температуре (2700K / 4000K / 6500K), световому потоку, классу защиты IP и типу монтажа (встраиваемый / накладной / подвесной).

### Подбор комплектующих

Светильник — это не единичный товар. К нему нужен драйвер, крепёж, иногда управляющий модуль. Когда платформа показывает совместимые комплектующие — это экономит время и снижает риск ошибки.
