## Специфика отрасли «Сетевая инфраструктура / Кабельная продукция»

Сетевая инфраструктура — отрасль с **огромным разнообразием стандартов и категорий**. От витой пары Cat5e до оптоволокна OM4, от коммутаторов до серверных шкафов — каждый товар должен соответствовать строгим стандартам совместимости.

В этой отрасли **ошибка в категории кабеля означает нерабочую сеть**. Инженеры проектируют СКС по строгой документации, и замена Cat6 на Cat5e на этапе монтажа — это переделка всего проекта.

## Ключевые вызовы дистрибуции сетевого оборудования

- **Комплектация объекта** — полный набор сетевого оборудования для нового объекта: кабель, коммутаторы, патч-панели, серверные шкафы
- **Заказ по проекту СКС** — подбор оборудования по проектной документации с точными стандартами и категориями
- **Модернизация** — замена или дополнение существующей инфраструктуры с сохранением совместимости
- **Аварийные закупки** — срочная замена вышедшего из строя оборудования
- **Расходные материалы** — регулярные поставки кабеля, коннекторов, патч-кордов

## Как платформа решает задачи поставщика сетевого оборудования

### Технические характеристики и стандарты

Каждый товар содержит полные характеристики: категория (Cat5e, Cat6, Cat6A), тип кабеля (витая пара, оптоволокно, коаксиальный), скорость, частота, тип разъёма. Соответствие стандартам ISO/IEC, TIA/EIA, ГОСТ указано в карточке.

### Совместимость оборудования

В карточке товара указаны совместимые товары — какое оборудование работает вместе. Это критически важно: коммутатор должен работать с определёнными SFP-модулями, кабель — соответствовать категории патч-панели.

### Поиск, заточенный под инженеров

Поиск по артикулу — основной сценарий, инженеры работают с конкретными моделями. Поиск по категории: «Cat6», «OM4», «SFP+» — типичные запросы. Поиск по типу кабеля: «витая пара», «оптоволокно». Фильтры по бренду, стандарту, типу разъёма, скорости и количеству портов.

### Документация для сдачи объекта

Протоколы испытаний и сертификаты соответствия доступны для скачивания в карточке каждого товара. Это обязательные документы для приёмки объекта и сертификации СКС.

### Гибкие единицы измерения

Кабель продаётся бухтами, метрами и на отрез. Платформа поддерживает все варианты с возможностью пересчёта, что устраняет путаницу при заказе.
