## Глоссарий иностранных слов и профессиональных терминов

*   **B2B-платформа:** Платформа оптовой торговли / программный комплекс для цифровизации оптовых продаж.
*   **SEO (Search Engine Optimization):** Действия, направленные на настройку и наполнение сайта для увеличения его видимости в поисковых системах.
*   **SPA-сайт (Single Page Application):** Одностраничное приложение.
*   **Рандомизатор метатегов:** Инструмент для автоматического формирования метаданных (заголовков и описаний).
*   **Server-Side Rendering (SSR):** Генерация страниц на стороне сервера.

## Дано: B2B Motion как SPA-сайт — преимущества и ограничения

Платформа B2B Motion — это SPA-приложение (Single Page Application), то есть одностраничный сайт со статическим URL без постоянных перезагрузок страниц. При первом посещении браузер загружает все статические данные. Динамический контент (навигация, фильтры каталога, обработка заказов) загружается с сервера и обновляется в ответ на действия пользователя. Это минимизирует нагрузку на сервер, что обеспечивает максимальную скорость загрузки страниц.

**Преимущества:**
*   Сверхбыстрая и плавная работа системы без долгих, раздражающих перезагрузок страниц
*   Время ожидания при прокрутке каталога менее 0,1 секунды
*   Критично для онлайн-магазинов с огромным ассортиментом и профессиональных покупателей
*   Клиенты могут платить, добавлять в корзину, возвращаться в каталог и искать гораздо быстрее, чем на классических многостраничных сайтах
*   Фокус на интуитивных интерфейсах и принципе «одного окна». Сложные задачи (подбор, заказ, спецификации) разбиты на простые шаги в рамках одной вкладки браузера

**Ограничения:**
*   Поисковые боты не могут качественно сканировать контент одностраничных приложений; им нужны статические страницы с персональными заголовками, описаниями и метатегами
*   HTML-документы SPA создаются динамически по запросу пользователя
*   Следовательно, B2B-система не отображалась в результатах поиска, и пользователи не могли найти ресурс через браузерные запросы

Раньше это не было проблемой, так как фокус был на «закрытом клубе для профессиональных оптовиков». Система была инструментом для установления связей с постоянными клиентами (персональные условия, цены, скидки), не предназначена для широкой аудитории. SEO-элементы не требовались, и сайт был закрыт для внешних посетителей.

## Задача: Внедрить SEO-инструменты для отображения в результатах поиска

Времена изменились, и B2B-рынок изменился; запросы на закрытые порталы стали неактуальны. Был запущен открытый каталог продуктов, позволяющий любому пользователю интернета выбирать товары, уточнять свойства, артикулы и цены без авторизации.

Клиенты запросили SEO, чтобы находить продукты B2B-системы через пользовательские запросы и обеспечивать качественную конверсию из поисковых систем без больших вложений в рекламу или масштабную разработку. Один клиент инвестировал во внедрение SEO-функционала, поставив следующие задачи:
*   Поисковый бот должен видеть все ссылки на страницы сайта
*   Контент каждой страницы должен быть доступен для индексации, включая заголовки, описания и другие метатеги, необходимые для поискового бота

## Поиск решений: SEO-инструменты от разработчиков Яндекс

Google довольно хорошо справлялся с индексацией SPA, видя веб-страницы аналогично пользователям. Яндекс — нет. На момент поиска решений Яндекс признавал проблемы с одностраничными сайтами и анонсировал бета-версию собственных технологий для индексации SPA.

В 2019 году Яндекс внедрил движки и алгоритмы для понимания контента SPA, запустив новые методы определения релевантности и инструменты индексации. Однако после трёх месяцев переписки с поддержкой, тестирования и наблюдения результаты поисковой оптимизации для B2B-платформы не изменились. Программисты продолжили поиски и потратили около 75 часов на исследования.

## Результат: Серверный рендеринг (SSR)

Программисты остановились на технологии серверного рендеринга, при которой HTML-код страницы генерируется на сервере, а не в браузере пользователя.

**Реализация:**
*   Поисковый бот перенаправляется с SPA-сайта на специальный сервер с заранее подготовленным статическим контентом
*   Формируется полная веб-страница с персональным URL, аналогично многостраничному сайту
*   Каждая страница уникальна и понятна боту

**Итог:**
*   Немедленно достигнуто 100 000 проиндексированных страниц в месяц для клиента
*   Разработан механизм обработки пустых страниц и ошибок 404, предотвращающий дублирование страниц, замедление индексации и увеличение нагрузки на сервер

**Основные преимущества SSR:**
*   **Мгновенная первая отрисовка:** HTML предварительно сформирован, поэтому пользователи видят контент почти сразу после загрузки
*   **Поддержка SEO:** Поисковые боты получают полностью оптимизированный контент с метатегами, заголовками и описаниями. Поисковые системы легко индексируют эти страницы и повышают релевантность сайта
*   **Совместимость:** Работает с устаревшими устройствами и браузерами. Снижает требования к вычислительным ресурсам устройства пользователя. Контент отображается корректно даже при отключённом JavaScript

## Собственная разработка: Рандомизатор метатегов

Для адаптации страниц B2B-платформы под поисковых ботов и упрощения внедрения SEO-инструментов для клиентов был разработан **Рандомизатор метатегов**.

*   **Функция:** Механизм на основе ИИ, который автоматически генерирует данные для поисковых ботов: заголовки, описания и другие метатеги для категорий, продуктов и брендов
*   **Уникальность:** Все данные уникальны и автоматизированы для каждой страницы
*   **Ручное редактирование:** SEO-редакторы могут вручную уточнять данные через новый раздел SEO в админ-панели
*   **Преимущество:** Заменяет крупномасштабный ручной труд, помогает внедрять SEO-инструменты и поднимает продукты в топ по среднечастотным запросам без дополнительных затрат на продвижение

**Пример результатов:**
Продукты клиента достигли топ-позиций в поиске по запросам вроде «растворимый кофе оптом в Уфе» и «печенье оптом Башкортостан».

## Технологии, внедрённые во время интеграции SSR

*   **TypeScript:** Перешли с JavaScript на TypeScript для улучшения качества и организации кода. Помогает выявлять ошибки до запуска программы
*   **Remix:** Полнофункциональный фреймворк для создания быстрых, надёжных, масштабируемых веб-приложений. (Примечание: Shopify купил Remix в 2022 году для перестройки инфраструктуры для более быстрой загрузки и лучшего поискового ранжирования)
*   **Web Vitals:** Подключены метрики Google, оценивающие скорость загрузки, интерактивность и визуальную стабильность, влияющие на поисковое ранжирование
*   **Tailwind:** Фреймворк, позволяющий изменять дизайн, не покидая HTML-разметки, обеспечивая пользовательские дизайны с большей гибкостью
*   **Яндекс.Вебмастер:** Подключён и настроен для мониторинга отображения и технического статуса B2B-платформы в поисковых системах

## Резюме

*   **Затраты:** Разработчики потратили 75 часов на исследование SEO-решений для SPA-сайтов и около 300 часов на детальную реализацию и адаптацию
*   **Индексация:** Поисковые боты просканировали более 100 000 страниц в первый месяц
*   **Ранжирование:** Продукты достигли топ-результатов поиска по среднечастотным запросам без затрат на рекламу
*   **Автоматизация:** Внедрено решение на основе ИИ для автоматического заполнения метатегов, заменяющее ручной труд (критично для больших ассортиментных матриц)
*   **Трафик:** Оптимизировано для поисковых систем; пользовательский трафик стал живым, кликабельность значительно выросла
*   **Доступность:** Это была масштабная модификация, заказанная и оплаченная клиентом. По принципам компании, новый функционал и расширенные возможности становятся доступны всем остальным клиентам по запросу без дополнительных затрат на разработку
