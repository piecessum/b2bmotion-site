## Специфика отрасли «Мебель и материалы для мебели»

Мебельная отрасль — это **сочетание плитных материалов и фурнитуры с точными характеристиками**. ЛДСП, МДФ, фанера имеют параметры толщины, формата, декора. Петли, направляющие, ручки — нагрузку, угол открывания, тип монтажа.

В этой отрасли **ошибка в 1 мм может испортить изделие**. Раскрой плитного материала в неправильном размере или несовместимая фурнитура — это брак всей партии.

## Ключевые вызовы поставок мебельных материалов

- **Заказ под проект** — комплектация мебельного проекта всеми материалами и фурнитурой
- **Повторные заказы** — стандартные позиции для серийного мебельного производства
- **Подбор фурнитуры** — подбор комплектующих для конкретного мебельного изделия с учётом нагрузки и типа монтажа
- **Раскрой плит** — заказ плитных материалов в нужных размерах, а не в стандартных форматах
- **Подбор декоров** — выбор конкретного декора ЛДСП/МДФ для проекта

## Как платформа решает задачи поставщика мебельных материалов

### Размеры, форматы и декоры

Для каждого плитного материала указаны точные габариты, доступные форматы раскроя и все варианты декоров с визуализацией. Закупщик видит «Дуб Сонома», «Бетон светлый» — и может выбрать нужный.

### Характеристики фурнитуры

Каждый элемент фурнитуры содержит полные характеристики: нагрузка, угол открывания, тип монтажа, размеры. Монтажные схемы и чертежи доступны для скачивания.

### Поиск, заточенный под мебельщиков

Поиск по артикулу — для точных заказов фурнитуры. Поиск по названию декора: «Дуб Сонома», «Бетон светлый». Поиск по типу фурнитуры: «петля с доводчиком», «направляющая скрытого монтажа». Фильтры по толщине, размеру, декору, типу фурнитуры, нагрузке.

### Совместимость

В карточке товара указаны совместимые товары: какая фурнитура подходит к каким плитам, какие крепежи нужны для конкретного материала. Это исключает ошибки при комплектации заказа.

### Раскрой

Возможность заказать материал в нужном размере — не только стандартные форматы, но и раскрой по размерам заказчика. Это критически важно для мебельных производств.
