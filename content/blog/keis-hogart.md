## Специфика отрасли «Сантехника и инженерное оборудование»

Хогарт работает на стыке двух направлений: **дизайнерская сантехника** и **технически сложное инженерное оборудование**. С одной стороны — внешний вид и коллекции, с другой — точные характеристики, мощность и производительность.

Более 200 брендов означают **огромное разнообразие товаров и спецификаций**. Закупщику нужна возможность быстро найти нужное оборудование среди тысяч позиций и быть уверенным в его характеристиках.

## Ключевые вызовы поставок инженерного оборудования

- **Комплектация объекта** — полный набор сантехники и инженерного оборудования для здания или комплекса
- **Заказ по проекту** — точные позиции из проектной документации с конкретными моделями
- **Подбор коллекции** — выбор единой дизайнерской серии для сантехники
- **Подбор аналогов** — замена недоступного оборудования с сохранением характеристик
- **Техническая документация** — паспорта, схемы, сертификаты для каждого товара

## Как платформа решает задачи поставщика инженерного оборудования

### Дизайнерские коллекции + технические характеристики

Для сантехники — качественные изображения, связь по коллекциям, визуальная комплектация. Для инженерного оборудования — полные технические характеристики: мощность, производительность, тип подключения, габариты.

### Документация в одном месте

Сертификаты соответствия, технические паспорта, схемы подключения — всё в карточке товара. Закупщик скачивает документы без обращения к менеджеру.

### Поиск по 200+ брендам

Поиск по артикулу — для точных заказов из проекта. Поиск по названию коллекции — для дизайнерской сантехники. Фильтры по бренду, типу оборудования, коллекции, техническим параметрам помогают быстро найти нужное среди тысяч позиций.

### Коммерческие предложения

КП формируется с изображениями сантехники и техническими характеристиками инженерного оборудования — для согласования с заказчиком, проектировщиком или дизайнером.
