#### Выбор региона

![](/wiki/function/lichnyy-kabinet-pokupatelya/02.webp)

> [!CALLOUT color=#4a9f2d]
> При входе в систему первым делом пользователь должен выбрать свой регион из доступного списка. Цены, наличие на складе и условия заказа привязаны к конкретному региону.

#### Личные данные

При необходимости пользователь может изменить свои данные (кроме сотового телефона) в личном кабинете. Личные данные можно посмотреть нажав на свою фамилию. Личные данные сюда подтягиваются те, которые пользователь вводил при регистрации.  
Здесь же пользователь может настроить комфортные для себя уведомления

![](/wiki/function/lichnyy-kabinet-pokupatelya/03.webp)

На этой странице так же находится блок изменения пароля

![](/wiki/function/lichnyy-kabinet-pokupatelya/04.webp)

#### Данные о компании

Данные о своей компании пользователь может посмотреть нажав в кабинете на слово "Покупатель"

![](/wiki/function/lichnyy-kabinet-pokupatelya/05.webp)

Здесь можно увидеть тип цены, который назначен для компании пользователя, кредитный лимит, если вы нам его передаете, а так же дебиторскую задолженность и просрочку.  
В режиме реального времени пользователь может отслеживать статусы платежей всех отгрузок по кампании. Отчет поделен на две части:  

1.  Просроченные платежи (с указанием дней просрочки)
2.  Непросроченные платежи (с указанием конечного срока оплаты)

Для удобства поиска нужного заказа все столбцы сортируются по возрастанию/убыванию.  
  
Все это стандарный функционал в2в системы и может подтягиваться по каждой компании из 1с.  
  
Так же пользователь здесь может увидеть контакты привязанного к нему менеджера и скорректировать адреса доставки

![](/wiki/function/lichnyy-kabinet-pokupatelya/06.webp)
