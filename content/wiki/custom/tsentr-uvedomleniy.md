Рассылки и отдельные сообщения для клиентов в B2B Маркете.  
Доработка платная, для уточнения стоимости, обратитесь к менеджеру.  
У клиента появится иконка колокольчика в шапке сайта, где хранятся уведомления системы

![](/wiki/custom/tsentr-uvedomleniy/01.webp)

Чтобы создать рассылку, перейдите в Админку → Центр уведомлений → Списки рассылки. Здесь можно создать список самостоятельно с помощью фильтров или загрузить готовый файл с телефонами пользователей.

![](/wiki/custom/tsentr-uvedomleniy/02.webp)

Для создания списка самостоятельно войдите в «Создать фильтр – список». Слева выберите необходимые фильтры для включения пользователей в список. Справа отобразится список пользователей, соответствующих заданным критериям.

![](/wiki/custom/tsentr-uvedomleniy/03.webp)

Для загрузки списка, перейдите в «Создать список».  
Загрузите файл в формате csv с одной колонкой — номерами телефонов пользователей, через 8 или +7 или 7.  
В список можно добавить пользователей из B2B. Список отражается справа.

![](/wiki/custom/tsentr-uvedomleniy/04.webp)

Чтобы создать рассылку, зайдите в Админка → Центр уведомлений → Рассылки → Добавить рассылку. Заполните строки внутри рассылки.

![](/wiki/custom/tsentr-uvedomleniy/05.webp)

> [!CALLOUT color=#4a9f2d]
> Что бы запустить рассылку выберите статус : Запуск

В строке «Список рассылки» выберите список, который будете отправлять. Тип — темы сообщений, которые создали: «Заказы» (для менеджеров и клиентов), «Скидки и акции», «Новые функции В2В», «Поддержка В2В», в интерфейсе у клиента темы отображаются отдельнеыми чипсами. Заполните контент: текст, ссылки, картинки, фотографии. Нажмите «Создать».

> [!CALLOUT color=#4a9f2d]
> Сейчас рассылки отправляются каждый час. Но тайминг можно изменить

Можно отправлять сообщения конкретному пользователю. Для этого зайдите в Админку → Центр уведомлений → Сообщения. Выберите получателя, тип, заголовок, контент и нажмите «Создать».

![](/wiki/custom/tsentr-uvedomleniy/06.webp)
